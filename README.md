#### Обоснование выбора набора данных для задачи классификации:  
Для задачи классификации используется набор данных Student Alcohol Consumption, который позволяет предсказать, сдаст ли студент итоговый экзамен. Этот выбор обоснован важностью прогнозирования академической успеваемости в образовательных учреждениях: модель поможет выявлять студентов в группе риска для своевременной поддержки, анализировать влияние социальных факторов на успехи в учебе и оптимизировать учебный процесс. Датированный, но качественный набор содержит 1044 наблюдения с 33 признаками (демографические, социальные, академические) и идеально подходит для бинарной классификации  

#### Обоснование выбора набора данных для задачи регрессии:  
Для задачи регрессии выбран датасет House Rent Prediction, который ставит целью прогнозирование ежемесячной арендной платы. Эта задача крайне востребована на рынке недвижимости: автоматическая оценка стоимости аренды полезна как для арендодателей и риелторов, так и для арендаторов, желающих проверить справедливость цены. Современный набор (2022 год) включает 4746 наблюдений и 12 признаков, описывающих характеристики жилья и местоположение. Оба датасета основаны на реальных данных, охватывают разные предметные области и предоставляют возможность реалистичного моделирования с измеримыми практическими результатами
